# Promo Codes & Discounts

**Article ID:** kb-015
**Last updated:** 2025-08-10

## Applying a promo code

Enter the code in the **Promo / Gift card** field at checkout and click **Apply**. The discount appears in your order summary.

## One code per order

Only **one promo code** can be applied per order. Gift cards do not count as promo codes and can be combined with one promo code.

## Common reasons a code is rejected

- **Expired.** Each code has its own validity window — check the email or campaign page you got it from.
- **Minimum order value not met.** Many codes require a minimum spend; the message at checkout will tell you the threshold.
- **Not valid for these items.** Some codes exclude sale items, gift cards, or specific brands.
- **First-time customer only.** Welcome codes can only be redeemed once per account / household.
- **Already redeemed.** Single-use codes cannot be reused.

## Forgot to apply a code

If you completed an order without applying a code, contact support within **48 hours** of placing the order with the code and your order number. We will retroactively apply it as a partial refund if the code was valid at the time of order. **After 48 hours we cannot apply codes retroactively.**

## Stacking discounts

Sale prices and subscription discounts are applied automatically and cannot be combined with most promo codes. The checkout will only let you apply codes that are compatible with what's in your cart.
