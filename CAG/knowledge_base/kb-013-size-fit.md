# Size & Fit Guidance

**Article ID:** kb-013
**Last updated:** 2025-07-12

## Size charts

Every clothing and footwear product page includes a **Size chart** link with measurements in both centimetres and inches. Charts are product-specific — a size M in one style is not always the same as size M in another.

## How to measure yourself

For best results:

1. Measure with a soft tape directly against your body (or over a thin layer of clothing).
2. Keep the tape level and snug, not tight.
3. Compare your measurements to the **garment size chart**, not a generic body chart.

Detailed measurement guides for chest, waist, hip, and inseam are linked from each size chart.

## Between sizes?

- For fitted styles, **size down**.
- For relaxed or oversized styles, **size up**.
- Each product page lists the model's height and the size they're wearing, which is the most reliable single signal.

## If the size is wrong

You can return clothing and footwear for a different size within the standard 30-day window — see *Return Policy*. We recommend ordering the new size as a fresh order rather than waiting for the return to process, since popular sizes can sell out.

## Footwear

Footwear runs in EU sizing by default. UK and US conversions appear on each product page. If you're between EU half sizes (e.g. EU 41.5), most of our styles do not stock half sizes — we recommend going up to the next full size.
